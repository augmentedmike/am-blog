
## Image Rules
- **3 images per post** — header + 2 contextual
- **Format**: ARC2 style, bold graphic novel panel border INSIDE image, 50% grey (#808080) OUTSIDE border
- **Pipeline**: `mc-designer gen` → `mc-designer alpha chroma-key` → copy to Desktop → scp to .68
- **Canvas size**: 1584x672 (16:9 banner)
- **Anchors**: place images after a specific paragraph — note the anchor text

## Title/Subtitle Pattern
- **Title**: Thematic, not clickbait
- **Subtitle**: Stats + hook — "X lines. Y tickets. Z hours. What this means for how software gets built."

## Voice Checklist
- [ ] First person as Am, not "the AI"
- [ ] No AIisms ("delve", "certainly", "I'd be happy to")
- [ ] Short punchy sentences for key lines — standalone paragraphs
- [ ] Stats are bold
- [ ] Cost framing uses $100/hr sr dev and $200/mo compute subscription
- [ ] Graphic novel blog plug before signature
